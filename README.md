# 🏫 ระบบสารสนเทศ วิทยาลัยพณิชยการบางนา

โปรเจกต์นี้เป็นส่วนหนึ่งของการสอบปฏิบัติมาตรฐานวิชาชีพ (ปวช. 3) สาขาเทคโนโลยีสารสนเทศ 
พัฒนาในรูปแบบ Single Page Routing (PHP) พร้อมดีไซน์สไตล์ Premium Minimalist รองรับการแสดงผลทุกหน้าจอ (Responsive)

## ✨ เทคโนโลยีที่ใช้ (Tech Stack)
- **Frontend:** HTML5, CSS3 (Custom Properties, Flexbox, Grid)
- **Backend:** PHP 8.x
- **Typography:** ฟอนต์ Prompt (Google Fonts)
- **Deployment:** Docker & Docker Compose

## 🚀 วิธีการติดตั้งและรันโปรแกรม (How to run)
โปรเจกต์นี้รองรับการรันผ่าน Docker เพื่อความสะดวกและไม่ต้องตั้งค่า Server บนเครื่องหลัก

1. ตรวจสอบให้แน่ใจว่าเครื่องของคุณติดตั้ง Docker และ Docker Compose แล้ว
2. เปิด Terminal และเข้าไปที่โฟลเดอร์โปรเจกต์
3. พิมพ์คำสั่งด้านล่างเพื่อรันเซิร์ฟเวอร์:
   ```bash
   docker-compose up -d
